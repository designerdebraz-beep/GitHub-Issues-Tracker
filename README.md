What is the difference between var, let, and const?

var is the old way to declare variables in JavaScript. It can be redeclared and updated.

let is the modern way to declare a variable. It can be updated but cannot be redeclared in the same scope.

const is used when the value should not change. It cannot be updated or redeclared.

What is the spread operator (...)?

The spread operator is used to copy values from an array or object.

It helps to merge arrays or copy data easily.

What is the difference between map(), filter(), and forEach()?

map() → creates a new array by changing each element.

filter() → creates a new array with selected elements based on a condition.

forEach() → only loops through the array but does not return a new array.
